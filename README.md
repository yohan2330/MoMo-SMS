## Setup
1. Clone repo: `git clone https://github.com/yohan2330/MoMo-SMS.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run ETL: `bash scripts/run_etl.sh`
4. Serve frontend: `python -m http.server 8000`
